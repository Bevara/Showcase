MP4 Demo

This page is embedded in the MP4 and opened by the service worker.

In this demo, we decode the mp4 file trailer_1080p.mp4 using the universal canvas tag :

<canvas is="universal-canvas_1" data-url="trailer_1080p.mp4" using="solver_with_sdl_1" with="isobmff_1;vorbis_1;theora_1" > < /canvas >
